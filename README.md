# 📅 Days Lived Calculator

A simple and stylish web application that calculates the total number of days you have lived based on your date of birth.

---

## 🚀 Features

- Select your date of birth using a date picker
- Instantly calculates total days lived
- Displays formatted result with commas (e.g., 10,245 days)
- Clean dark-themed UI
- Responsive centered layout

---

## 🛠️ Technologies Used

- HTML5 – Structure of the webpage  
- CSS3 – Styling with dark theme and modern layout  
- JavaScript (Vanilla JS) – Date calculation logic
